# addUrlProps([options])

Higher order component (HOC) that injects URL query parameters as props.

#### Arguments

1. [`options`] (*Object*)
  * mapUrlToProps
  * mapUrlChangeHandlersToProps
  * urlPropsQueryConfig
  * addRouterParams
  * addUrlChangeHandlers
  * changeHandlerName

#### Returns

A React component class

#### Remarks

* Needs to be called twice `addUrlProps(options)(MyComponent)`.
